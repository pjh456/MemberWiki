# 系统架构说明

本项目采用前后端分离架构：React + FastAPI + SQLite（可选 Redis）+ Nginx + Docker。

## 架构分层

1. 展示层（Frontend）

- React + TypeScript 页面与组件
- 通过 REST API 调用后端服务

2. 应用层（Backend API）

- FastAPI 路由、鉴权、参数校验
- 用户、百科、审核、搜索、上传等业务逻辑

3. 数据层（Database）

- SQLite 存储核心业务数据
- ORM 映射与查询
- 按并发压力可选引入 Redis 缓存

4. 交付层（Ops）

- Nginx 反向代理与静态资源分发
- Docker/Compose 打包与部署

## 关键业务流

1. 注册登录：注册 -> 登录 -> 获取 JWT -> 访问受保护接口。
2. 百科编辑：编辑资料 -> 保存草稿 -> 提交审核 -> 管理员审批。
3. 搜索检索：关键词检索 -> 多字段查询 -> 分页返回结果。

## 非功能要求

- 安全：权限校验、输入校验、上传限制、敏感字段脱敏。
- 性能：分页、索引优化、缓存按需接入。
- 可维护：接口先行、文档先行、迁移可回滚。
